CreateDTPicker (create a datetimepicker control)

     CreateDTPicker( LONG param_Left , LONG param_Top, LONG param_Width, LONG param_Height, UNSIGNED param_Handle, BOOL param_bFocus )

CreateDTPicker            Creates a DateTimePicker control with the specified dimensions.

 

param_Left                  Specifies the initial horizontal position of the control. param_Left is the x-coordinate of the upper-left corner of the control relative to the upper-left corner of the parent window's client area.

  

param_Top                  Specifies the initial vertical position of the control. param_Top is the initial y-coordinate of the upper-left corner of the control relative to the upper-left corner of the parent window's client area.

  

param_Width                  Specifies the width, in device units, of the control.

  

                       If param_Width is 0 (zero), the param_Width is defaulted to 200.

  

param_Height                  Specifies the height, in device units, of the control.

 

                       If param_Height is 0 (zero), the param_Height is defaulted to 10.

 

param_Handle                  Specifies the Window Handle of the control that the DateTimePicker control will follow in the dialog tabbing order.

 

param_bFocus                  If TRUE, the control will be given focus.

  

Return Value                  True.

 

The following style properties can be set prior to calling this method:

  • AppCanParse

  • DisplayFormat

  • ShowNone

  • UpDown

  • RightAlign
