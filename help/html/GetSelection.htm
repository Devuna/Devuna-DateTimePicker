GetSelection (get currently selected date/time)

     GetSelection( [*LONG param_lDateValue] [,*LONG param_lTimValue] )

GetSelection                  Retrieves the currently selected date or time from a DateTimePicker control.

 

param_lDateValue            pointer to a Clarion LONG variable type to receive the currently selected date in Clarion format.

 

param_lTimeValue            pointer to a Clarion LONG variable type to receive the currently selected time in Clarion format.

 

Return Value                  Returns GDT_VALID if the date/time information was successfully placed in param_lSelection. Returns GDT_NONE if the control was set to the DTS_SHOWNONE style and the control check box was not selected. Returns GDT_ERROR if an error occurs.

 

                       The following EQUATEs are defined in DateTimePick.inc:

 

                       GDT_ERROR      EQUATE(-1)

                       GDT_VALID      EQUATE(0)

                       GDT_NONE      EQUATE(1)
