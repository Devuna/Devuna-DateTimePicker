UpDown (display spinner)

     UpDown            BOOL

 

           The UpDown property controls the up-down (spin) control style. When enabled, an up-down control is placed to the right of the DateTimePicker control to modify date-time values. This style can be used in place of the drop-down month calendar, which is the default style.

 

           Set the UpDown property by assigning one of the following values:

 

           FALSE            The drop-down month calendar will be used to select the date.

 

           TRUE            The up-down control will be used to select the date.

 

           The class user should call the CreateDTPicker or Refresh methods after setting this property.
