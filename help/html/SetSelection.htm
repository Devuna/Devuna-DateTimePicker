SetSelection (set currently selected date/time)

     SetSelection( [LONG param_lDateValue] [,LONG param_lTimeValue] )

SetSelection                  Sets the date or time in a DateTimePicker control.

 

param_lDateValue            a Clarion LONG variable containing the Clarion Date for the control.

 

param_lTimeValue            a Clarion LONG variable containing the Clarion Time for the control.

 

Return Value                  Returns nonzero if successful, or zero otherwise.
