SetFormat (set display format and refresh)

     SetFormat( LONG param_lFormat )

SetFormat                  Sets the DisplayFormat property of the DateTimePicker control based on a given format and redisplays the control.

 

param_lFormat                  The format to be used.

 

           The param_lFormat parameter must be set to one of the following EQUATEs defined in DateTimePick.inc:

 

           DTPLONGDATE      EQUATE(0)

           Displays the date in long format. The default format string for this style is defined by LOCALE_SLONGDATE, which produces output like "Friday, April 19, 1996".

 

           DTPSHORTDATE EQUATE(1)

           Displays the date in short format. The default format string for this style is defined by LOCALE_SSHORTDATE, which produces output like "4/19/96".

 

           DTPTIME EQUATE(2)

           Displays the time. The default format string for this style is defined by LOCALE_STIMEFORMAT, which produces output like "5:31:42 PM".

 

           DTPCUSTOM EQUATE(3)

           Sets the display of a DateTimePicker control based on the CustomFormat property.

 

     Be sure to set the CustomFormat property using the SetCustomFormat method before calling SetFormat with the DTPCUSTOM format.

 

 

Return Value                  non-zero if successful or zero otherwise.
