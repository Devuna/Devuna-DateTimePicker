KCR_DateTimePicker extension template

The KCR_DateTimePicker template adds a DateTimePicker control to the window.

 

Place and size a rectangle on the window to specify the position of the control.

 

 

ControlName            Specify a unique name to use to reference this instance of the DateTimePicker class.

 

AppCanParse            Set this check box if your application needs to parse the user input.

 

Parse Function            Specify the name of a procedure that will be called in response to the DTN_USERSTRING notification event.

  This procedure must be prototyped as follows:

 

     MyParseFn      PROCEDURE(*CSTRING szUserString, *LONG lDateValue, *LONG lTimeValue),ULONG

 

                 The procedure should parse the szUserString parameter and fill in the values for lDateValue and lTimeValue.

 

Remember that this procedure is called as part of an event notification process; the procedure should execute quickly and not require any user interaction.

 

                 The procedure must return one of the following EQUATEs as defined in DateTimePicker.inc:

 

                       GDT_ERROR      EQUATE(-1)

                       GDT_VALID      EQUATE(0)

                       GDT_NONE      EQUATE(1)                  

 

Format Option            Select one of the radio button choices. The choices and their meaning are as follows:

 

Choice                  Meaning

Long Date            Displays the date in long format. The default format string for this style is defined by LOCALE_SLONGDATE, which produces output like "Friday, April 19, 1996".

 

Short Date            Displays the date in short format. The default format string for this style is defined by LOCALE_SSHORTDATE, which produces output like "4/19/96".

 

Time                  Displays the time. The default format string for this style is defined by LOCALE_STIMEFORMAT, which produces output like "5:31:42 PM"..

 

Custom                  Sets the display of a DateTimePicker control based on the CustomFormat property.

 

Custom Format      A DateTimePicker format string.

 

ShowNone            Sets the ShowNone property for the control. Enabling this option causes a check box to appear as part of the DateTimePicker control.

 

UpDown            Sets the UpDown property for the control. Enable this option to use a spin control to select the date.

 

RightAlign            Sets the RightAlign property for the control. Enable this option to right align the drop-down calendar with the DateTimePicker control.

 

Visible                  Sets the Visible attribute for the control. Enable this option to have the DateTimePicker control initially hidden.

 

Focus                  Forces focus to the control when the window is opened. The control will be in the same tab order as the defining rectangle. If the control is the first control in the tab order, the Clarion runtime library will not give it focus since rectangles cannot receive focus. In this circumstance, check this attribute to give the control focus.

 

Rectangle            Select the control to be used to place and size the DateTimePicker control. Normally this would be a rectangle that you have placed on the window to represent the location and size of the DateTimePicker control.

 

Date Field            This is a Clarion LONG variable that will be used to hold the selected date value.

 

Time Field            This is a Clarion LONG variable that will be used to hold the selected time value.

 

Change Notification Event      Specify an event number or EQUATE that will be posted to your window when the DateTimePicker value changes.

 

This template works by responding to DateTimePicker control messages.
