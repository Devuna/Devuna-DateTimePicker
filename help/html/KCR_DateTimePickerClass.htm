KCR_DateTimePickerClass extension template

This APPLICATION level template adds the necessary equates and libraries to your application to enable you to utilize the DateTimePicker class.

 

You must add this template to your application global extensions before you can use any of the other DateTimePicker templates.

 

There are no prompts for this template.

 

The template:

  • adds comctl32.lib to your project

  • includes DateTimePick.inc

  • adds prototypes for the CallWindowProc and memcpy api functions
