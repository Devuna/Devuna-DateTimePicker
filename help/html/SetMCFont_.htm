SetMCFont (set month calendar font)

     SetMCFont( UNSIGNED param_hFont, BOOL param_fRedraw )

SetMCFont                  Sets the font to be used by the DateTimePicker control's child month calendar control.

 

param_hFont                  Handle to the font that will be set.

 

param_fRedraw                  Specifies whether the control should be redrawn immediately upon setting the font. Setting this parameter to TRUE causes the control to redraw itself.

 

Return Value                  Returns an HFONT value that is the handle to the previous font.
