KCR_SubClassClientWindow extension template

The KCR_SubClassClientWindow PROCEDURE extension template creates a procedure and subclasses the client window area to allow processing of DateTimePicker control events.

 

There are no prompts for this template.

 

The template:

  • Creates a MODULE level variable to store the original ClientWindowProc

  • Adds a new ClientWindowProc to the module map

  • Adds the new ClientWindowProc to the module source file

  • Subclasses the ClientWindowProc in the Window.Init procedure

  • Removes the subclassed ClientWindowProc in the Window.Kill procedure

 

You must add this template to your PROCEDURE level extensions before you can use the KCR_DateTimePicker template.
