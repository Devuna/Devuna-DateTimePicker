GetRange (get allowable range)

     GetRange( [*LONG param_lDateMinimum] [,*LONG param_lTimeMinimum] [, *LONG param_lDateMaximum] [, *LONG param_lTimeMaximum] )

GetRange                  Retrieves the current minimum and maximum allowable dates/times for a DateTimePicker control.

 

param_lDateMinimum            A Clarion LONG variable that is to receive the minimum date value if set. If this parameter is omitted then no minimum value is returned.

 

param_lTimeMinimum            A Clarion LONG variable that is to receive the minimum time value if set. If this parameter is omitted then no minimum value is returned.

 

param_lDateMaximum            A Clarion LONG variable that is to receive the maximum date value if set. If this parameter is omitted then no maximum value is returned.

  

param_lTimeMaximum            A Clarion LONG variable that is to receive the maximum time value if set. If this parameter is omitted then no maximum value is returned.

 

Return Value                  Returns a ULONG value that is a combination of GDTR_MIN or GDTR_MAX. The param_lMinimum contains the minimum allowable date/time if GDTR_MIN is set. The param_lMaximum contains the maximum allowable time if GDTR_MAX is set.

 

                       The following EQUATEs are in DateTimePick.inc:

                       GDTR_MIN      EQUATE(00001h)

                       GDTR_MAX      EQUATE(00002h)
