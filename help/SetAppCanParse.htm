SetAppCanParse (set the AppCanParse property)

     SetAppCanParse( BOOL param_bFlag )

SetAppCanParse            Sets the AppCanParse property for the control.

 

param_bFlag                  This parameter specifies the new value for the AppCanParse property.

 

The param_bFlag parameter must be set to one of the following:

 

     Value                  Meaning

     FALSE                  Do not allow owner to parse user input. Do not send DTN_USERSTRING notification messages

 

     TRUE                  Allow owner to parse user input. Send DTN_USERSTRING notification messages
