LongToSystemTime

     LongToSystemTime( LONG param_lTime, *tagSYSTEMTIME param_systime )

LongToSystemTime            Utility method to return a Clarion LONG time from the passed SystemTime structure.

 

param_lTime                  A Clarion LONG time value that is to be converted and stored in the SystemTime structure.

 

param_systime                  This parameter is the SystemTime structure that will receive the converted time value.

 

Return Value                  None.
