SetRange (set allowable range)

     SetRange( [LONG param_lDateMinimum] [,LONG param_lTimeMinimum] [, LONG param_lDateMaximum] [, LONG param_lTimeMaximum] )

SetRange                  Sets the minimum and maximum allowable dates/times for a DateTimePicker control.

 

param_lDateMinimum            A Clarion LONG variable that contains the minimum date value to set. If this parameter is omitted then no minimum value is set.

 

param_lTimeMinimum            A Clarion LONG variable that contains the minimum time value to set. If this parameter is omitted then no minimum value is set.

 

param_lDateMaximum            A Clarion LONG variable that contains the maximum date value to set. If this parameter is omitted then no maximum value is set.

 

param_lTimeMaximum            A Clarion LONG variable that contains the maximum time value to set. If this parameter is omitted then no maximum value is set.

  

Return Value                  Returns nonzero if successful, or zero otherwise.
