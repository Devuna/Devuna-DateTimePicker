DisplayFormat (display format)

     DisplayFormat            LONG

 

           The DisplayFormat property indicates the format string to use when invoking the SetFormat method.

 

           Set the DisplayFormat property by assigning one of the following EQUATEd values. The following EQUATEs are in DateTimePick.inc:

 

           DTPLONGDATE      EQUATE(0)

           Displays the date in long format. The default format string for this style is defined by LOCALE_SLONGDATE, which produces output like "Friday, April 19, 1996".

 

           DTPSHORTDATE EQUATE(1)

           Displays the date in short format. The default format string for this style is defined by LOCALE_SSHORTDATE, which produces output like "4/19/96".

 

           DTPTIME EQUATE(2)

           Displays the time. The default format string for this style is defined by B>LOCALE_STIMEFORMAT, which produces output like "5:31:42 PM".

 

           DTPCUSTOM EQUATE(3)

           Sets the display of a DateTimePicker control based on the CustomFormat property.
