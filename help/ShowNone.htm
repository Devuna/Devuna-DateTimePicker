ShowNone (display checkbox)

     ShowNone            BOOL

 

           The ShowNone property controls the processing of null dates.

 

           It is possible to have no date currently selected in the control. With this style, the control displays a check box that users can check once they have entered or selected a date. Until this check box is checked, the application will not be able to retrieve the date from the control because, in essence, the control has no date.

 

           Set the ShowNone property by assigning one of the following values:

 

           FALSE            The check box will not be displayed

 

           TRUE            The check box will be displayed

 

           The class user should call the CreateDTPicker or Refresh methods after setting this property.
