DateTimePicker

DateTimePicker Class API and Template Specification

 

The following help[ file contains descriptions of the methods defined in the DateTimePicker class and descriptions of the templates provided to simplify the use of the class.

 

DateTimePicker provides a Clarion wrapper (DateTimePicker.clw) for the DateTimePicker Common Control located in the Windows Common Control Library (comctl32.dll).

 

To use the DateTimePicker class in your application follow these simple steps:

 

  1. Add the KCR_DateTimePickerClass Template extension template to the Application Global Extensions.

  2. Place and size a Rectangle Control on the window where you want the DateTimePicker Common Control to appear. In the Properties dialog on the General tab, set the Mode to Hide.

  3. Add the KCR_SubClassClientWindow extension template to the method containing the window on which you wish to use the DateTimePicker common control.

  4. Add the KCR_DateTimePicker extension template to the method containing the window on which you wish to use the DateTimePicker common control.

 

A sample program DateTimePicker.app is provided that demonstrates use of all of the templates supplied with the DateTimePicker Class and Templates.
