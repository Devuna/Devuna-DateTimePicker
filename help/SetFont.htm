SetFont (set control font)

     SetFont( *CSTRING param_szTypeFace, LONG param_lFontSize , LONG param_lFontStyle )

SetFont                  Sets the font to be used by the DateTimePicker control.

 

param_szTypeFace            A string variable containing the name of the font. If blank, the 'MS San Serif' font is used.

 

param_lFontSize            An integer constant or variable containing the size (in points) of the font. If 0, an 8 point font size is used.

 

param_lFontStyle            An integer constant, constant expression, EQUATE, or variable specifying the strike weight and style of the font.

 

Return Value                  Returns an HFONT value that is the handle to the previous font.
