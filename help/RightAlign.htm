RightAlign (right align dropdown)

     RightAlign            BOOL

 

           The RightAlign property controls the alignment of the drop-down month calendar.

 

           Set the RightAlign property by assigning one of the following values:

 

           FALSE            The drop-down month calendar will be left-aligned

 

           TRUE            The drop-down month calendar will be right-aligned

 

           The class user should call the CreateDTPicker or Refresh methods after setting this property.
