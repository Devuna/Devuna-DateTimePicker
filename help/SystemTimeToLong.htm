SystemTimeToLong (get long time from systemtime)

     SystemTimeToLong( tagSYSTEMTIME param_systime )

SystemTimeToLong            Utility method to return a Clarion LONG time from the passed SystemTime structure.

 

param_systime                  This parameter is the SystemTime structure that contains the time value to be extracted.

 

Return Value                  Returns a Clarion LONG time.
