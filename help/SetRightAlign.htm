SetRightAlign (set RightAlign property)

     SetRightAlign( BOOL param_bFlag )

SetRightAlign            Sets the RightAlign property for the control.

 

param_bFlag                  This parameter specifies the new value for the RightAlign property.

 

The param_bFlag parameter must be set to one of the following:

 

     Value            Meaning

     FALSE            The drop-down month calendar will be left-aligned

 

     TRUE            The drop-down month calendar will be right-aligned
