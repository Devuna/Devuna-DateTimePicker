SetUpDown (set UpDown property)

     SetUpDown( BOOL param_bFlag )

SetUpDown                  Sets the UpDown property for the control.

 

param_bFlag                  This parameter specifies the new value for the UpDown property.

 

The param_bFlag parameter must be set to one of the following:

 

     Value            Meaning

     FALSE            The drop-down month calendar will be used to select the date.

 

     TRUE            The up-down control will be used to select the date.
