Init (initialize the DateTimePicker class)

     Init()

Init                        Initializes the DateTimePicker Class. This method must be called before making any other calls to the class.

 

                       This method registers specific common controls classes from the common control dynamic-link library (comctl32.dll) and initializes properties private to the class.

 

Return Value                  None.
