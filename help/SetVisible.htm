SetVisible (hide or unhide control)

     SetVisible( BOOL param_bFlag )

SetVisible                  Hides or Unhides the control.

 

param_bFlag                  This parameter specifies the new visible state for the control.

 

     The param_bFlag parameter must be set to one of the following:

 

           Value            Meaning

           FALSE            the control is hidden

           TRUE            the control is visible

 

 

Return Value                  Returns the previous state of this property.
