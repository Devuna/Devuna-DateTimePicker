SetShowNone (set ShowNone property)

     SetShowNone( BOOL param_bFlag )

SetShowNone            Sets the ShowNone property for the control.

 

param_bFlag                  This parameter specifies the new value for the ShowNone property.

 

The param_bFlag parameter must be set to one of the following:

 

     Value            Meaning

     FALSE            The check box will not be displayed

 

     TRUE            The check box will be displayed
