SetCustomFormat (sets the CustomFormat property)

     SetCustomFormat( *CSTRING param_szCustomFormat )

SetCustomFormat            Set the CustomFormat property of the control. This property is used by the SetFormat method when the Format property of the control is set to DTPCUSTOM.

 

param_szCustomFormat      Address of a zero-terminated format string that defines the desired display. Setting this parameter to NULL will reset the control to the default format string for the current style.

 

Return Value                  None.
