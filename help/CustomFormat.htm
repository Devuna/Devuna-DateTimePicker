CustomFormat (custom format string)

     CustomFormat            CSTRING(81)

 

           The CustomFormat property specifies the format string to use when invoking the SetFormat method when the DisplayFormat property is set to DTPCUSTOM.

 

           It is acceptable to include extra characters within the format string to produce a more rich display. However, any nonformat characters must be enclosed within single quotes. For example, the format string "'Today is: 'hh':'m':'s ddddMMMdd', 'yyy" would produce output like "Today is: 04:22:31 Tuesday Mar 23, 1996".

           

     A DateTimePicker control tracks locale changes when it is using the default format string. If you set a custom format string, it will not be updated in response to locale changes.
