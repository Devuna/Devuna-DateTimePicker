Visible (show control)

     Visible            BOOL

 

           The Visible property controls the visibility of the DateTimePicker control.

 

           Set the Visible property by assigning one of the following values:

 

           FALSE            The control is not visible.

 

           TRUE            The control is visible.

 

           The class user should call the CreateDTPicker or Refresh methods after setting this property.
